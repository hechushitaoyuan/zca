后台鉴权密钥与网关访问控制
鉴权
用于登录此管理后台。修改后需用新密码重新登录。
设置后，调用 /v1/messages 须携带 Authorization: Bearer <key> 或 x-api-key。留空则不校验。
后台自动刷新各账号额度与状态的周期。设为 0 关闭自动刷新（仍可手动刷新）。修改后即时生效。
使用说明
· 在「账号池」粘贴 Coding Plan JWT 或 API Key 即可加入轮询。
· 请求按 round-robin 分发；某账号额度用完会自动切到下一个账号。
· 账号额度、状态在「账号池」页实时刷新展示。
· 对话端点：/v1/messages（兼容 Anthropic Messages 协议）。
